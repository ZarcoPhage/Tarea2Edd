Tarea 2 - Problema 2: RankedABB
=============================================

--------------------------------------------------------------------
REQUERIMENTOS DEL AMBIENTE PARA COMPILAR
--------------------------------------------------------------------
- Sistema operativo: Linux, distribución preferiblemente basada en Debian
- Compilador g++

--------------------------------------------------------------------
COMPILACIÓN
--------------------------------------------------------------------
- Comando de compilacion:
  $ g++ main.cpp -o test -Wall

- genera el archivo 'test' en el directorio Tarea2Edd/problema2
--------------------------------------------------------------------
EJECUCIÓN
--------------------------------------------------------------------
- luego de compilar el código main.cpp, ejecutar el comando:
  $ ./test

- si se desea utilizar otro archivo main.cpp, leer en detalle los comentarios y mode de uso de las funciones en el archivo:
  /ranked-abb.cpp
